Customize the default Gravity Forms with a more modern design and layout.

== Description ==

**Description:** Welcome to Modern Designs for Gravity Forms: Easily transform your forms to a modern looking style and layout. You can use a Responsive Grid to layout your fields in a more comfortable format. Field and label heights will be aligned with each other and UI elements like Dropdowns, Checkboxes, and Radios with be styles to a more modern design.

==Requirements==

- Gravity Forms
- jQuery
- WordPress 4.5 or above


== Installation ==

1. Upload the `gravityfroms-modern-designs` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can configure the Global Settings in `Forms` -> `Settings` -> `Modern Designs`
